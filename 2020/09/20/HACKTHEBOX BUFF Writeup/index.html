靶机信息

Buff是HACKTHEBOX平台上的一台靶机，信息如下:

OS:Windows
Difficulty:	Easy
Points:	20
Release:18 Jul 2020
IP:	10.10.10.198

其中有20分，分别是Own User和Own Root

Own User

Type below the hash that is inside the user.txt file in the machine. 
The file can be found under /home/{username} on Linux machines and
at the Desktop of the user on Windows.

Own Root

Type below the hash that is inside the root.txt file in the machine. 
The file can be found under /root on Linux machines and at the Desktop
of the Administrator on Windows.
